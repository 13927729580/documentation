[[installation]]
== Installation

[[setup_addon]]
=== Setup CUBA BI Add-on

. Clone BI Add-on from https://git.haulmont.com/platform/bi.git

. Open BI Add-on project in CUBA Studio and install https://doc.cuba-platform.com/manual-6.4/app_components_sample.html[app component]: *Run - Install app component*

[[connect_addon]]
=== Add BI to your application

. Connect the app component to your project

* Open your project in CUBA Studio

* Edit *Project properties* and on the *App components* panel click the *plus* button next to *Custom components*

* In the *Custom application component* dialog, select the `bi` project in the *Registered project* drop-down list. The list contains all project registered in Studio that have an `app-component.xml` descriptor.
+
image::bi_component.png[align="center"]

. Add BI component to the screens

* By default, the BI add-on creates new menu item *BI* with the screens to create and run Pentaho report at a runtime. Click *BI -> Reports -> Create* and set the path to your Pentaho report:
+
image::bi_create.png[align="center", height="200"]

* The second option is to add manually the _bi_ element to the xml-descriptor of the screen:
+
--
[source,xml]
----
include::{sourcesdir}/bi_component.xml[]
----

The `reportPath` is the path to the report file in Pentaho User Console. Add the `cubabi` namespace to the `xmlns` attribute of the `window` element:

[source, xml]
----
xmlns:cubabi="http://schemas.company.com/cubabi/0.1/ui-component.xsd"
----
