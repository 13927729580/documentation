== Polymer Core

https://www.polymer-project.org[Google Polymer] is a JavaScript library that allows to create reusable components.

CUBA UI module relies on Polymer. So, it's important to understand basics of this framework.

Google Polymer has an https://www.polymer-project.org/2.0/docs/devguide/feature-overview[official development guide].
However, to expedite and facilitate the learning we created our own quick manual that accents the most important
features of the library and provides a number of examples to better understand how various features can be used.

If you need to know more after our course we encourage you to use the
https://www.polymer-project.org/2.0/docs/devguide/feature-overview[official manual].

It's useful to know that Polymer currently has 2 major versions: 1.0 and 2.0. Their syntax is very different. All our
examples use Polymer 2.0.
