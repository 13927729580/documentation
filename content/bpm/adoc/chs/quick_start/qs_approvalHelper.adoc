=== 创建 ApprovalHelper Bean

将在合同审批流程中调用 `ApprovalHelper` bean 的 `updateState()` 方法来设置合同状态。

方法参数：


* `entityId` - 合同实体标识符


* `state` - 合同状态

在 IDE 中打开项目。在 `com.company.bpmdemo.core` 包中创建 `ApprovalHelper` 类。
