[[intro]]
== Introduction

BPM module complements the CUBA platform with an ability of execution business processes described in BPMN 2.0 notation. Activiti framework (http://www.activiti.org) is used for business process execution. Besides an integration with Activiti the BPM module gives a visual process editor.

[[studio-support]]
== Enabling the BPM module support in Cuba Studio

At present Studio support for BPM module is disabled. To enable it you have add the next code to %USERPROFILE%\.haulmont\studio\config.xml:

.config.xml
[source,xml]
----
<bpm>
    <enabled>true</enabled>
</bpm>
----

[[app-properties]]
== Application properties

bpm.activiti.asyncJobExecutorEnabled:: Possible values: true or false. Defines whether Job Executor for timers and asynchronous tasks is enabled. Default value is false.
