[[studio-support]]
== Enabling the BPM module support in Cuba Studio

At present Studio support for BPM module is disabled. To enable it you have add the next code to %USERPROFILE%\.haulmont\studio\config.xml:

.config.xml
[source,xml]
----
<bpm>
    <enabled>true</enabled>
</bpm>
----

[[app-properties]]
== Application properties

bpm.activiti.asyncJobExecutorEnabled:: Possible values: true or false. Defines whether Job Executor for timers and asynchronous tasks is enabled. Default value is false.
