===== 孤立网络使用 CUBA Studio

如果要在孤立网络使用 CUBA Studio，按照下面这些步骤：

- 下载 https://services.gradle.org/distributions/[Gradle] (需要下载相应的 {gradleVersion} 版本) 然后在开发机器安装。
- 打开 CUBA Studio 服务窗口。
- 点击 *Advanced* 按钮并且设置安装的 Gradle 目录。
- 启动 Studio 服务。
- 按照<<private_repo_usage,上面提到的>>步骤配置项目。
