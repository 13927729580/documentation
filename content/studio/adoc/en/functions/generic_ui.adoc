[[generic_ui]]
=== Generic UI

The {main_man_url}/gui_framework.html[Generic UI] of the project consists of application menu, screens, themes and messages. They are displayed in the *Generic UI* section of the tree.

* <<generic_ui_menu,Web menu>> allows you to open the menu editor of the corresponding module.

* {main_man_url}/main_message_pack.html[Main Message Pack] section contains messages for main menu entries and common UI elements names.

* <<generic_ui_screens,Screens>> section displays the created application screens.

* <<generic_ui_themes,Themes>> are used to manage the visual presentation of the application.
